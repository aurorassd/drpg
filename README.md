# Auto探索ローグライク (JavaScript)

オート探索型のローグライクです。プレイヤーは行動方針を選ぶだけで、キャラクターが自動でダンジョンを探索します。

## 実装済み要件
- 2x2以上のランダム部屋 + 1マス幅通路で毎回異なるダンジョンを生成
- 階段・アイテム・モンスター配置
- プレイヤー1マス移動ごとにモンスターも1マス移動
- 接触時にターン制バトル（プレイヤー→モンスター交互）
- 撃破時に4択ステータス強化（STR/DEF/SPD/LUK）
- ダメージ式: `与ダメ = 総合攻撃力 * 防御力補正`
- 防御力補正: `max(0.1, 1 - 総合防御力/300)`
- 物理命中率: `(1 + (攻撃側SPD - 防御側SPD)/100)*100` を 5〜100% にクランプ

## ローカルでの起動
```bash
npm run dev
```
`http://localhost:4173` を開いてプレイできます。

## テスト
```bash
npm test
```
(Node.js 標準の `node:test` を使用)

## GitHub にアップロードして公開（GitHub Pages）
1. GitHubでリポジトリ作成
2. このプロジェクトを push
3. `Settings > Pages` で GitHub Actions を有効化
4. 同梱の `.github/workflows/pages.yml` で自動公開
